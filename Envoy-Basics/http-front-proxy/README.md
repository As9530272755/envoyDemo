# HTTP Front Proxy demo

### 环境说明
三个Service:
- envoy：Front Proxy,地址为172.31.2.2
- webserver01：第一个后端服务,地址为172.31.2.11
- webserver02：第二个后端服务,地址为172.31.2.12

### 运行和测试
1. 创建
```
docker-compose up
```

2. 测试
```
curl -H "host: www.ik8s.io" 172.31.2.2
curl -I -H "host: www.magedu.com" 172.31.2.2
```

3. 停止后清理
```
docker-compose down
```
